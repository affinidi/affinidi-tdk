# ServiceEndpointDto

Service endpoint information

## Properties

| Name            | Type       | Description                                        | Notes      |
| --------------- | ---------- | -------------------------------------------------- | ---------- |
| **id**          | **String** | service endpoint ID                                | [optional] |
| **name**        | **String** | name of the service endpoint                       | [optional] |
| **description** | **String** | description of the service endpoint                | [optional] |
| **url**         | **String** | service endpoint URL                               | [optional] |
| **walletAri**   | **String** | wallet ARI this endpoint belongs to                | [optional] |
| **projectId**   | **String** | project ID                                         | [optional] |
| **createdAt**   | **String** | when this endpoint was created                     | [optional] |
| **modifiedAt**  | **String** | when this endpoint was last modified               | [optional] |
| **createdBy**   | **String** | identifier of the user who created the entity      | [optional] |
| **modifiedBy**  | **String** | identifier of the user who last updated the entity | [optional] |
